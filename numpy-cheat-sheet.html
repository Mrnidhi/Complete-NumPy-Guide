Ultimate NumPy Cheat Sheet

A comprehensive guide to NumPy functions and operations for data analysis and scientific computing

Basics

The fundamental functions to get started with NumPy arrays, including creation methods and basic operations. 
import numpy as np                    # Import the NumPy library
np.array([1, 2, 3])                # Create a 1D array
np.zeros((2, 3))                   # Create a 2x3 array of zeros
np.ones((2, 3))                    # Create a 2x3 array of ones
np.full((2, 3), 7)                 # Create a 2x3 array filled with 7s
np.eye(3)                          # Create a 3x3 identity matrix
np.arange(0, 10, 2)                # Create array with values 0,2,4,6,8
np.linspace(0, 1, 5)               # Create array with 5 evenly spaced values from 0 to 1
np.random.rand(2, 3)               # Create a 2x3 array with random values (0-1)
np.random.randn(2, 3)              # Create a 2x3 array with random values from standard normal distribution
np.random.randint(0, 10, (2, 3))   # Create a 2x3 array with random integers from 0 to 9
np.empty((2, 3))                   # Create an uninitialized 2x3 array
np.copy(arr)                       # Create a copy of an array

Array Attributes

Properties and methods to access information about NumPy arrays such as shape, size, and data type. 
arr.shape        # Dimensions of the array (rows, columns)
arr.ndim         # Number of dimensions
arr.size         # Total number of elements
arr.dtype        # Data type of elements
arr.itemsize     # Size in bytes of each element
arr.nbytes       # Total size in bytes of the array
arr.T            # Transpose of the array
arr.real         # Real part of complex array
arr.imag         # Imaginary part of complex array

Indexing, Slicing, Iterating

Methods to access and manipulate specific elements or subsets of arrays, including advanced indexing techniques. 
arr[1]                  # Get second element (0-indexed)
arr[1:4]                # Get elements from index 1 to 3
arr[:, 0]               # Get first column
arr[1, 2]               # Get element at row 1, column 2
arr[-1]                 # Get last element
arr[::2]                # Get every other element
arr[::-1]               # Get all elements in reverse order
arr[1, ::2]             # Get every other element from the second row
arr[..., 1]             # Get second column using ellipsis notation
np.take(arr, [0, 2])    # Take elements at indices 0 and 2
np.put(arr, [0, 2], [9, 9])  # Put values 9 at indices 0 and 2
for x in np.nditer(arr):     # Iterate through all elements
    print(x)

Data Types

NumPy supports various data types for efficient memory usage and computation, allowing precise control over data representation. 
arr.astype(float)       # Convert array to float type
np.int32                # 32-bit integer type
np.int64                # 64-bit integer type
np.float32              # 32-bit floating point type
np.float64              # 64-bit floating point type
np.bool_                # Boolean type
np.complex_             # Complex number type
np.dtype('int32')       # Create a data type object

Math Operations

Perform element-wise mathematical operations on arrays, including arithmetic, trigonometric, and exponential functions. 
arr + 5                    # Add 5 to each element
arr - 5                    # Subtract 5 from each element
arr * 2                    # Multiply each element by 2
arr / 2                    # Divide each element by 2
arr ** 2                   # Square each element
np.add(arr1, arr2)         # Element-wise addition
np.subtract(arr1, arr2)    # Element-wise subtraction
np.multiply(arr1, arr2)    # Element-wise multiplication
np.divide(arr1, arr2)      # Element-wise division
np.mod(arr1, arr2)         # Element-wise modulus
np.remainder(arr1, arr2)   # Element-wise remainder
np.exp(arr)                # e raised to power of each element
np.sqrt(arr)               # Square root of each element
np.log(arr)                # Natural logarithm of each element
np.log10(arr)              # Base-10 logarithm of each element
np.sin(arr)                # Sine of each element
np.cos(arr)                # Cosine of each element
np.tan(arr)                # Tangent of each element
np.arcsin(arr)             # Inverse sine of each element
np.arccos(arr)             # Inverse cosine of each element
np.arctan(arr)             # Inverse tangent of each element
np.radians(arr)            # Convert degrees to radians
np.degrees(arr)            # Convert radians to degrees
np.clip(arr, a_min, a_max) # Clip values between a_min and a_max

Aggregation Functions

Functions to compute summary statistics and aggregate data across arrays, such as mean, sum, minimum, and maximum. 
np.sum(arr)             # Sum of all elements
np.mean(arr)            # Mean of all elements
np.std(arr)             # Standard deviation
np.var(arr)             # Variance
np.min(arr)             # Minimum value
np.max(arr)             # Maximum value
np.argmin(arr)          # Index of minimum value
np.argmax(arr)          # Index of maximum value
np.ptp(arr)             # Range of values (max - min)
np.percentile(arr, 50)  # 50th percentile (median)
np.median(arr)          # Median value
np.cumsum(arr)          # Cumulative sum
np.cumprod(arr)         # Cumulative product

Broadcasting

NumPy's broadcasting mechanism allows operations between arrays of different shapes, automatically expanding dimensions when needed. 
arr + 5                    # Add 5 to each element
arr + np.array([1, 2, 3])   # Add corresponding values from array [1,2,3] to each row

Logical Operations

Functions to perform logical operations on arrays, including comparison, boolean operations, and condition testing. 
arr > 0                             # Return boolean array for condition
arr == 5                            # Check equality for each element
np.where(arr>0, arr, 0)             # Return arr where condition is True, else 0
np.any(arr > 0)                     # True if any element satisfies condition
np.all(arr > 0)                     # True if all elements satisfy condition
np.logical_and(arr1 > 0, arr2 > 0)  # Element-wise AND of two conditions
np.logical_or(arr1 > 0, arr2 < 0)   # Element-wise OR of two conditions
np.logical_not(arr)                 # Element-wise NOT (inverting boolean array)
np.isfinite(arr)                    # Test which elements are finite
np.isnan(arr)                       # Test which elements are NaN
np.isinf(arr)                       # Test which elements are infinite

Sorting, Searching, Counting

Functions to sort and search arrays, find unique values, and perform set operations between arrays. 
np.sort(arr)                 # Return sorted copy of array
np.argsort(arr)              # Return indices that would sort array
np.lexsort((arr1, arr2))     # Sort by multiple keys (arr2, then arr1)
np.searchsorted(arr, 5)      # Find indices where elements should be inserted to maintain order
np.count_nonzero(arr)        # Count non-zero elements
np.unique(arr)               # Return sorted unique elements
np.in1d(arr1, arr2)          # Test whether elements in arr1 are in arr2
np.intersect1d(arr1, arr2)   # Find common elements in both arrays
np.setdiff1d(arr1, arr2)     # Find elements in arr1 that are not in arr2
np.union1d(arr1, arr2)       # Find union of elements from both arrays

Reshaping, Combining, Splitting

Functions to change array shapes, stack multiple arrays together, or split arrays into smaller segments. 
arr.reshape(2, 3)                      # Reshape array to dimensions (2,3)
arr.flatten()                          # Flatten multi-dimensional array to 1D (copy)
arr.ravel()                            # Flatten array (view if possible)
arr.T                                  # Transpose array
np.vstack([arr1, arr2])                # Stack arrays vertically
np.hstack([arr1, arr2])                # Stack arrays horizontally
np.concatenate([arr1, arr2], axis=0)   # Concatenate arrays along first axis
np.split(arr, 3)                       # Split array into 3 equal parts
np.array_split(arr, 3)                 # Split array into 3 parts (handles uneven splits)
np.hsplit(arr, 2)                      # Split array horizontally into 2 parts
np.vsplit(arr, 2)                      # Split array vertically into 2 parts
np.expand_dims(arr, axis=0)            # Add new axis at position 0
np.squeeze(arr)                        # Remove single-dimensional entries
np.tile(arr, (2, 2))                   # Construct array by repeating arr in a 2x2 pattern
np.repeat(arr, 2)                      # Repeat elements of array

Linear Algebra

Functions for performing linear algebra operations on arrays, including matrix multiplication, decomposition, and solving linear systems. 
np.dot(arr1, arr2)           # Dot product of arrays
np.matmul(arr1, arr2)         # Matrix multiplication
np.linalg.inv(arr)            # Compute inverse of a matrix
np.linalg.det(arr)            # Compute determinant of a matrix
np.linalg.eig(arr)            # Compute eigenvalues and eigenvectors
np.linalg.eigh(arr)           # Eigenvalues and eigenvectors of Hermitian matrix
np.linalg.svd(arr)            # Singular value decomposition
np.linalg.solve(A, b)         # Solve linear system Ax = b
np.trace(arr)                 # Sum of diagonal elements
np.linalg.norm(arr)           # Calculate norm of matrix or vector

Random

Functions for generating random numbers and performing random sampling from various probability distributions. 
np.random.seed(0)                    # Set random seed for reproducibility
np.random.rand(2, 3)                 # Random values between 0-1 in a 2x3 array
np.random.randn(2, 3)                # Random values from standard normal distribution
np.random.randint(0, 10, (2, 3))     # Random integers from 0-9 in a 2x3 array
np.random.choice(arr, size=3)        # Randomly select 3 elements from array
np.random.shuffle(arr)               # Shuffle array in-place
np.random.permutation(arr)           # Return a shuffled copy of array

Statistics

Functions for computing statistical measures on arrays, such as mean, median, correlation, and covariance. 
np.mean(arr)                # Calculate mean of all elements
np.median(arr)              # Calculate median of all elements
np.var(arr)                 # Calculate variance
np.std(arr)                 # Calculate standard deviation
np.percentile(arr, 25)      # Calculate 25th percentile (first quartile)
np.corrcoef(arr1, arr2)     # Calculate correlation coefficient
np.cov(arr1, arr2)          # Calculate covariance matrix

Missing Values

Functions for handling missing or invalid data (NaN values) in NumPy arrays, allowing for robust calculations. 
np.isnan(arr)              # Test element-wise for NaN
np.isfinite(arr)           # Test element-wise for finite numbers
np.nan_to_num(arr)         # Replace NaN with zero and infinity with large numbers
np.nanmean(arr)            # Calculate mean ignoring NaNs
np.nansum(arr)             # Calculate sum ignoring NaNs
np.nanstd(arr)             # Calculate standard deviation ignoring NaNs
np.nanmin(arr)             # Find minimum value ignoring NaNs
np.nanmax(arr)             # Find maximum value ignoring NaNs

File I/O

Functions for reading and writing NumPy arrays to different file formats, including text, binary, and compressed formats. 
np.loadtxt('file.txt')                         # Load data from text file
np.savetxt('file.txt', arr)                    # Save array to text file
np.genfromtxt('file.csv', delimiter=',')      # Load data from CSV file with flexible handling
np.save('file.npy', arr)                      # Save array to .npy binary file
np.load('file.npy')                           # Load array from .npy file
np.savez('file.npz', arr1=arr1, arr2=arr2)    # Save multiple arrays to .npz file
np.load('file.npz')                           # Load multiple arrays from .npz file

Special Functions

Specialized functions for working with arrays, including mesh grid generation and operations on diagonal or triangular matrices. 
np.meshgrid(x, y)           # Create coordinate matrices from coordinate vectors
np.mgrid[0:5, 0:5]          # Create dense meshgrid with even spacing
np.ogrid[0:5, 0:5]          # Create open meshgrid for efficient computations
np.diag([1,2,3])            # Create diagonal matrix or extract diagonal
np.tril(arr)                # Get lower triangular part of a matrix
np.triu(arr)                # Get upper triangular part of a matrix
np.trace(arr)               # Sum along the diagonal
